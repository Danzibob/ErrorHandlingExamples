## Installation and Configuration
Setup (from ./Python):
```bash
python3 -m venv env
source env/bin/activate
pip3 install couchbase
```
Configuration:

Run script (from ./Python):
```bash
python3 main.py
```
Leave the virtual environment:
```bash
deactivate
```
Re-enter the virtual environment (./Python):
```bash
source env/bin/activate
```
